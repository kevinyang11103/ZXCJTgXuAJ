# 城市交通管理系统技术文档 java33

## 项目概述

城市交通管理系统是一款基于SpringBoot和Vue的前后端分离系统。该系统致力于为管理员和用户提供便捷的城市交通管理和服务功能。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 系统环境
- 兼容Windows和macOS操作系统

### 推荐运行环境
- JDK 1.8
- Maven
- MySQL
- Node.js

### 服务端技术
- SpringBoot
- MyBatis-Plus

### 前端技术
- Vue
- Axios
- Element-UI
